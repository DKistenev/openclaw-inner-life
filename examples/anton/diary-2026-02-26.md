## What I Learned
- ZAI endpoint paths are fragile. One wrong segment gives a cryptic numeric
  error code (1113) with zero context about what's actually wrong. Documented
  the correct path in MEMORY.md so this won't happen again.
- The v4 deployment went smoother than expected. Having a written plan with
  13 numbered tasks made execution predictable — no decisions needed mid-flight.
- `gateway.env` can't be `source`d directly because it has non-bash syntax.
  Need `grep -E '^KEY_' | sed 's/^/export /'` pipeline.
